# Control Users on your Computer script
This Script allows you to control and add new users to tammoOS

Installationguide for older tammoOS versions:
1. Open the terminal
2. Download the Script: git clone https://github.com/tammoOS/script
3. navigate to the Script with: cd script
4. Type: chmod +x User\ Settings.sh
5. Done :)

Run the script with: sudo ./User\ Settings.sh
